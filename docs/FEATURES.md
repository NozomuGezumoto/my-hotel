# アプリ機能一覧（Project My Hotel）

高級ホテル（一泊8万円以上）の「行った・行きたい」管理と、夢の貯金・達成を追うアプリの全機能です。

---

## 1. ナビゲーション・基本

| 項目 | 内容 |
|------|------|
| **タブ** | Map / Brands / Countries / Dream（4タブ） |
| **多言語** | 日本語・英語（端末ロケールで自動切り替え） |
| **永続化** | 行った・行きたい・Dream・メモ・写真などは AsyncStorage で保存 |
| **データソース** | ローカル JSON（`luxury_hotels.json`）。取得は `scripts/fetch_luxury_hotels.js` で Amadeus から一泊8万円以上を取得 |

---

## 2. Map タブ（世界地図）

- **世界地図**で高級ホテルをピン表示（クラスタリング対応）
- **フィルター**
  - ステータス: すべて / 行った / 行きたい
  - 地域: アジア・ヨーロッパ・北米・中南米・オセアニア・アフリカ
  - 国: 地域に応じた国一覧
  - 都道府県・州: 国選択時に表示（日本は都道府県、海外は州・地域）
- **ボトムシート**
  - ピンタップでホテル詳細をシート表示
  - 一覧表示モードでフィルター結果のホテルリストを表示
- **リスト項目**に「行った」「行きたい」バッジ表示
- ピンタップまたはリストタップで **ホテル詳細画面** へ遷移

---

## 3. Brands タブ（ブランド一覧）

- **ブランド別**にホテル数を集計し、達成度を表示
- 各ブランド: **行った数 / 総数・達成率（%）・Dream 登録数**
- ブランドタップで **ブランド詳細**（そのブランドのホテル一覧）
  - 達成度プログレスバー
  - **ブランド称号**: 探索者（1件）・トラベラー（半分）・マスター（全件）
  - ホテル一覧（行った/行きたいバッジ付き）→ タップでホテル詳細

---

## 4. Countries タブ（国別達成）

- **国別**にホテル数・達成度を表示
- 各国: **行った数 / 総数・達成率（%）**
- 国タップで **国詳細**
  - 達成度プログレスバー
  - **国称号**: 探索者・トラベラー・マスター
  - ホテル一覧 → タップでホテル詳細

---

## 5. Dream タブ（叶えたい夢・叶えた夢）

- **Achieved dreams（叶えた夢）** — 1セクションに2種を統合
  - **夢で訪れた**: 夢に追加していたホテルを「行った」にしたもの  
    - 表示: 夢が始まった日・達成した日（日付は2行で表示）
  - **貯金目標達成**: 目標金額まで貯まったもの（「目標達成」バッジ）
- **叶えたい夢（Dreams in progress）**
  - 夢に追加したが未達成のホテル
  - カード: ホテル名・場所・価格・**励み用写真**（1枚・追加/変更/削除）
  - **貯金目標・期限・貯金額**の入力
  - 進捗バー・残り日数・月あたり必要額の表示
- 夢に追加した日（登録日）は「叶えた夢」カード内で「夢が始まった日」として表示
- 空のときの案内メッセージ・ヒント表示

---

## 6. ホテル詳細画面（共通）

Map / Brands / Countries / Dream のいずれかから **ホテルをタップ**で表示。

### 6.1 基本情報

- ホテル名・ブランド・場所・住所・緯度経度
- 一泊料金（通貨はロケールに応じて 円/¥, $ など）
- 概要（description）があれば表示
- **行った / 行きたい** タグ・トグル（行ったにする / 行ったを取り消す、夢に追加 / 夢から外す）

### 6.2 写真（Photos）

- 最大4枚まで追加（カメラロールから選択）
- 削除時は確認ダイアログ

### 6.3 満足度（Rating）

- **行った**にしているホテルのみ表示
- 1〜5の星で評価

### 6.4 思い出（Memories）

- 自由メモ（複数行）

### 6.5 Dream Plan

- **夢に追加**: 未追加ならボタンで追加（初期目標額＝一泊料金）
- **夢に追加済み**の場合:
  - 期限・貯金目標・貯金額の入力
  - 進捗・残り日数・月あたり必要額の表示
  - 夢から外す

### 6.6 Stay Record（宿泊記録）

- **行った**にしているホテルのみ
- 宿泊日（日付）の入力・表示
- 「行ったを取り消す」で未訪問に戻せる

### 6.7 外部リンク

- 地図アプリで開く（Google Maps / Apple Maps）
- Webで検索（ホテル名で検索）

## 7. ストア（状態）で管理している主なデータ

| データ | 説明 |
|--------|------|
| `visitedHotels` | 行ったにしたホテル ID 一覧 |
| `wantToGoHotels` | 行きたいにしたホテル ID 一覧 |
| `hotelVisitedDates` | ホテルごとの宿泊日（行ったにした日またはユーザー入力） |
| `hotelDeadlines` | Dream の期限 |
| `hotelSavingGoals` | Dream の貯金目標額 |
| `hotelSavedAmounts` | Dream の貯金額 |
| `hotelMemos` | メモ・写真（最大4枚） |
| `hotelRatings` | 満足度 1〜5 |
| `dreamHotelImages` | Dream 用の励み画像（ホテルごと1枚） |
| `dreamVisitedHotelIds` | 夢に追加していたが「行った」にした ID（Achieved dreams 用） |
| `dreamAddedDates` | 夢に追加した日（登録日） |

※ Shop 関連の状態（visitedShops, wantToGoShops 等）はストアに定義ありだが、現在の UI では未使用。

## 8. その他

- **通貨表示**: 日本ロケールでは「円」、英語では「¥」など、ロケールに応じた表記
- **ホテル詳細**: ScrollView + SafeAreaInsets + KeyboardAvoidingView でスクロール・キーボード対応
- **地図**: カスタムスタイル・クラスタリング・ボトムシートでピン一覧・詳細表示
